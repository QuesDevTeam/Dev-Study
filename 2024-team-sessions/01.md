# Environment Variables
## How It Works
- Open one of these files
```shell
$ vim ~/.bashrc
$ vim ~/.bash_profile
$ vim ~/.zshrc
```

## In Practice
- Create script
```vim
#!/bin/sh

echo $ASDF
```

- Test
```console
$ ./env_test.sh
(no output)

$ ASDF=12 ./env_test.sh
12

$ export ASDF=33

$ ./env_test.sh
33
```
